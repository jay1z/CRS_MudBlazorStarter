# Specialist Guide

Welcome to the Specialist Guide! This documentation will help you manage your assigned reserve studies efficiently from proposal to report delivery.

---

## Your Role

As a **Specialist (TenantSpecialist)**, you are the hands-on professional conducting reserve studies:

| Responsibility | Description |
|----------------|-------------|
| ✅ Create proposals | Draft service agreements for clients |
| ✅ Collect data | Request and review financial information |
| ✅ Conduct site visits | Inspect community properties |
| ✅ Assess components | Evaluate condition and remaining life |
| ✅ Generate reports | Create comprehensive reserve study documents |
| ✅ Submit for QA | Ensure quality before admin review |

## Workflow Overview

Your typical workflow for each study:

```
Study Assigned → Create Proposal → Send to Client
                                         ↓
                               Proposal Accepted
                                         ↓
                          Request Financial Info → Review Data
                                         ↓
                          Schedule Site Visit → Conduct Inspection
                                         ↓
                          Upload Photos/Data → Configure Funding
                                         ↓
                          Write Narrative → Preview Report
                                         ↓
                          Quality Check → Submit for QA
                                         ↓
                          (Admin Publishes) → Complete
```

## Quick Reference

| Task | Where to Go |
|------|-------------|
| View my studies | Dashboard → My Work |
| Create a proposal | Study → Create Proposal |
| Request financial info | Study → Financial Info → Request |
| Schedule site visit | Study → Schedule Visit or Calendar |
| Upload photos | Study → Photos → Upload |
| Edit components | Study → Components |
| Generate report | Study → Report → Preview |
| Submit for review | Study → Submit for QA |
