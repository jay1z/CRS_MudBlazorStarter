# Admin Guide

Welcome to the Admin Guide! As a TenantOwner (Admin), you have full control over your organization's reserve study operations, user management, and system configuration.

---

## Your Role

As an **Admin (TenantOwner)**, you are responsible for:

| Responsibility | Description |
|----------------|-------------|
| ✅ User management | Create and manage staff and HOA users |
| ✅ Workflow oversight | Approve requests, assign specialists |
| ✅ Quality assurance | Review and approve draft reports |
| ✅ Publishing | Deliver final reports to clients |
| ✅ Settings | Configure system-wide defaults |
| ✅ Financials | Manage invoices and payments |
| ✅ Reporting | Monitor business performance |

## Key Workflows

### Your Decision Points

```
Request Submitted → You Review & Approve
                           ↓
                   Assign Specialist
                           ↓
            Specialist Works (Proposal → Site Visit → Report)
                           ↓
               Report Submitted for QA
                           ↓
               You Review & Approve Report
                           ↓
               You Publish to Client → Complete
```

---

## Quick Reference

| Task | Where to Go |
|------|-------------|
| Add a user | Tenant Admin → Manage Users |
| Approve a request | Dashboard → Pending Requests |
| Assign a specialist | Study → Assign Specialist |
| Review a report | Dashboard → QA Queue |
| Publish a report | Study → Publish |
| Create an invoice | Invoices → Create |
| View aging report | Reports → A/R Aging |
| Change settings | Tenant Admin → Settings |
